# Инструкция по запуску сервера

## Способ 1: Через bat-файл (рекомендуется)
Дважды кликните на файл `start-server.bat` в корне проекта.

## Способ 2: Через командную строку
Откройте командную строку или PowerShell в папке проекта и выполните:
```
php artisan serve
```

## Способ 3: С указанием хоста и порта
```
php artisan serve --host=127.0.0.1 --port=8000
```

После запуска сервер будет доступен по адресу:
- http://localhost:8000
- http://127.0.0.1:8000

**Важно:** Оставьте окно командной строки открытым, пока работает сервер. Для остановки нажмите Ctrl+C.
